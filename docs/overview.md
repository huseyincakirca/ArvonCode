# ArvonCode – Overview

ArvonCode, NFC tabanlı dijital kimlik ve araç yönetimi sistemidir.

Amaç:
- Fiziksel NFC kart ile dijital kimliği bağlamak
- Mobil uygulama üzerinden mesaj, konum ve durum yönetimi yapmak
- Güvenli ve ölçeklenebilir bir altyapı kurmak

Bu proje:
- Flutter mobil uygulama
- Laravel API
- NFC + konum + mesaj entegrasyonu
üzerine kuruludur.
