# ArvonCode – Proje Genel Bakış

ArvonCode, NFC/QR tabanlı araç kimliklendirme ve iletişim sistemidir.

Amaç:
- Araç sahibine ulaşmayı **numara paylaşmadan** mümkün kılmak
- Ziyaretçilerin (guest) araçla ilgili mesaj, konum ve uyarı bırakabilmesini sağlamak
- Araç sahibinin tüm bu verileri tek panelden yönetmesi

Sistem üç ana bileşenden oluşur:
- Mobile App (Flutter)
- Backend API (Laravel)
- NFC / QR fiziksel etkileşim

Bu klasördeki tüm dokümanlar **gerçek çalışan API kodlarından türetilmiştir**.
Kod neyse dokümantasyon odur.
